# All In Wallet

**All In Wallet** is a Python-based financial program developed as a **Harvard CS50 Final Project**.

The application provides real-time cryptocurrency and foreign exchange conversion, tracks money transactions through simulated real-world banking systems, validates credit card numbers using the **Luhn Algorithm**, and displays all completed operations through a simple user interface.

## Features
- Real-time cryptocurrency conversion
- Foreign exchange (FX) conversion
- Transaction tracking and logging
- Credit card number validation using the Luhn Algorithm
- User interface displaying completed operations

## Technologies Used
- Python
- External APIs (cryptocurrency & exchange rate data)
- File handling for transaction and wallet tracking
